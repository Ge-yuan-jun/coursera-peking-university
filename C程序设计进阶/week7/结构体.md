
#### 一、结构体
> 一种新的数据类型，属于组合数据类型，用一组变量来描述一件事物

```c++
struct student {
  int id;
  char name[20];
  char sex;
  int age;
  float score;
  char addr[30];
};  // 注意最后的‘;’
```
类似于js中的对象

在声明类型的同时定义变量

```c++
struct student {
  int id;
  char name[20];
  char sex;
  int age;
  float score;
  char addr[30];
}a, b;
```
