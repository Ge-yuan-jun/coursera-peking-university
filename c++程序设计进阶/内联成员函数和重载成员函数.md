### 内联成员函数
1. `inline` + 成员函数
2. 整个函数体出现在类定义内部

内联成员函数的两种定义方式：
```
class B {
  inline void func1(); // 1. 函数定义在类的外部
  void func2() {       // 2. 函数定义在类的内部

  };
};
void B::func1(){}
```

### 成员函数的重载及参数缺省
1. 重载成员函数
2. 成员函数-带缺省参数
```
#include<iostream>
using namespace std;
class Location {
  private:
    int x, y;
  public:
    void init(int x = 0, int y = 0);
    // 以下两个函数是成员函数重载
    void valueX(int val) {x = val;}
    int valueX() {return x;}
}
void Location::init(int X, int Y) {
  x = X;
  y = Y;
}
int main() {
  Location A;
  A.init(5); // X=5 Y=0
  A.valueX(5); // valueX(int val)
  cout << A.valueX(); // valueX()
  return 0;
}
```

使用缺省参数注意避免有函数重载时的二义性
```
class Location {
  private:
    int x, y;
  public:
    void init(int x = 0, int y = 0);
    void valueX(int val = 0) {x = val;}
    int valueX() {return x;}
}
```
